<h2>config üìù</h2>
</div>
<div align="left">
To begin config use <code>vim</code> or <code>nvim</code> to open the <code>config.py</code>. or if your lazy run the following
<pre>
cd ~/tbhfetch
nano config.py
</pre>
change it to either:
 - mac
 - arch
 - linux
if you chose linux type your distro in the <code>linuxdistro</code> variable.
